## Steps to start app

- `update composer`
- `sudo npm install`
- Create datatabase
- Set .env file
    - DB_DATABASE
    - DB_USERNAME
    - DB_PASSWORD
    - BROADCAST_DRIVER
    - PUSHER_APP_ID
    - PUSHER_APP_KEY
    - PUSHER_APP_SECRET
    - PUSHER_APP_CLUSTER
    - PAYPAL_SANDBOX_CLIENT_ID
    - PAYPAL_SANDBOX_SECRET
    - PAYPAL_LIVE_CLIENT_ID
    - PAYPAL_LIVE_SECRET
    - PAYPAL_MODE
- `php artisan key:generate`
- `php artisan storage:link`
- `php artisan migrate:refresh --seed`
- `php artisan passport:install`
